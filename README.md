webfile
===========================

Compile unix command file to js, and make it a webpage.

Use [emscripten](https://github.com/kripken/emscripten/wiki).

nodejs
---------------------------

run local:

    make node_file
    
result:

    node file.js test/a.out
    ELF 32-bit LSB executable, Intel 80386, version 1 (SYSV)
